# indecission-app
Random Decission Maker
Remover Dependencias: 
>> yarn remove babel-cli live-server 
>> npm uninstall -g babel-cli live-server
Prerequisites: Node JS
To Install:
>> yarn install
To run in two consoles:
>> yarn run dev-server
